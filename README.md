## ‚ú® The Experience

This is not just a collection of links; it's a journey. The core concept revolves around **"Shifting the Realm"**, where users can toggle between distinct visual and auditory states (Light & Dark Realms), each with its own personality and atmospheric depth.

### üé≠ Key Features
- **Cinematic Realm Transitions**: Smooth, synchronized changes in lighting, typography, and audio that redefine the user's environment in 800ms.
- **3D Celestial Environments**: Built with **Three.js** and **React Three Fiber**, featuring dynamic particle systems and interactive orbs that pulse with life.
- **Immersive Audio-Visuals**: Integrated ambient soundtracks that react to the realm state, creating a deeply atmospheric experience.
- **Typography & Glitch Aesthetics**: Custom **Decrypted Text** and **Glitch** effects that give the interface a mysterious, high-tech feel.
- **Performance Optimized**: Leveraging **Vite** and **React 19** for lightning-fast loads and buttery-smooth 60fps animations.

## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sreyoshi244/portfolio_sreyoshi2026.git
   cd portfolio_sreyoshi2026
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
